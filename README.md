## Development 


Install Android Studio

	http://developer.android.com/sdk/index.html

Install Android SDK & buildtools

	Android 6.0 (API 23)
	buildtools 23.0.1 and above

Open workspace

	Android Studio -> File -> Open

Run Unit Tests

	$ ./gradlew build

Build project

	$ gradlew build
    
# Libraries used:

*	RxAndroid

*	Retrofit

*	OkHttp

*	Dagger2

*	Mockito

*	Timber

*	Expresso

*	GSON

    
## Connect to local service    

```
├── README.md
├── clients/
|   ├── android/
│   └── ios/
├── people.raml
├── res/
│   └── images/
└── servers/
    └── people/
        ├── images/
        └── ppl_server.py
```

-   Make sure you have Python 3.5 or higher version installed

-   Install python packages using `pip` and `requirements.txt` file.

```
$ cd PROJECT_DIR
$ cd servers/people
$ pip install -r requirements.txt
```

### Running People Server

```
$ cd PROJECT_DIR
$ cd servers/people
$ python3 ppl_server.py

```

You should see output as follow:

